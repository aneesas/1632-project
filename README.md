## Requirements
- ffmpeg (for generating animation)
- YAPSS

### Installing ffmpeg
MacOSX does not ship with ffmpeg installed by default. The easiest way to get ffmpeg is via Homebrew:

```bash
brew install ffmpeg
```

If on Linux, similarly use apt:

```bash
sudo apt install ffmpeg
```
